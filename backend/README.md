# ERP Cosmetics Backend

Backend service untuk platform ERP Penjualan Kosmetik & Obat Tradisional Berbasis Trend.

## Tech Stack

- **Language**: Go 1.23.1
- **Framework**: Gin
- **ORM**: GORM
- **Database**: MySQL 8.0+
- **Cache**: Redis 7+
- **Auth**: JWT
- **Payment**: Midtrans
- **Shipping**: RajaOngkir

## Prerequisites

- Go 1.23.1 or higher
- MySQL 8.0+
- Redis 7+
- Make (optional)
